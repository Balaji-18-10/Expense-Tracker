# Expense Tracker 💰

A simple and efficient Expense Tracker web application built to help users manage their daily income and expenses with ease.

## Features ✨

- Add income and expense transactions
- Delete transactions
- Real-time balance calculation
- Track total income and total expenses
- Transaction history management
- Data stored using browser Local Storage
- Responsive and clean user interface

## Tech Stack 🛠️

- HTML5
- CSS3
- JavaScript
- Local Storage

## Installation 🚀

Clone the repository:

```bash
git clone https://github.com/Balaji-18-10/Expense-Tracker.git
```

Navigate to the project folder:

```bash
cd Expense-Tracker
```

Open the project:

Simply open `index.html` in your browser.

## Project Structure 📂

```bash
Expense-Tracker/
│── index.html
│── style.css
│── script.js
│── README.md
```

## How It Works ⚙️

1. Add income or expense details.
2. View your current balance instantly.
3. Track all transactions in the history section.
4. Data remains saved using browser Local Storage even after refresh.
